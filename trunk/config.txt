# DVD AUTOMATED MENU GENERATOR SETTINGS:
#---------------------------------------
# The following have the form: PrefixSuffix[:<menu-page-number>]
# Prefix is TOC for main-menu pages
# Prefix is CHP for submenu pages
# Suffix can be either: topleft, topright, bottomleft or bottomright.
# TOC suffix can also be leftbuttons or rightbuttons to limit the number of buttons on a page.
# Suffixes ending with "T" are menu targets which override the default target.
# Note: All UI strings may include special codes which will be replaced by values at runtime:
#   %0$S = becomes current book
#   %1$S = becomes current chapter
#   %2$S = becomes current verse
#   %3$S = becomes current last-verse

#TOCbottomleft:1  = About this DVD

TOCtopright:1     = Table of Contents
CHPtopleft        = %0$S
